# Portfolio personnel

## Description

Portfolio personnel développé en PHP vanilla permettant de présenter mon parcours, mes compétences techniques et mes projets.

## Technologies utilisées

- **Backend** : PHP (sans framework)
- **Frontend** : HTML5, CSS3, JavaScript vanilla
- **Bibliothèques** : AOS (Animate On Scroll)

## Contexte

Ce projet a été réalisé avant mon apprentissage des frameworks frontend modernes (React, Vue.js). Il est donc resté en PHP vanilla.

## Fonctionnalités

- Présentation du parcours académique et professionnel
- Catalogue des compétences techniques (langages, outils, frameworks)
- Vitrine des projets personnels avec captures d'écran
- Design responsive adapté aux différents supports
- Animations au défilement
